# Personal Portfolio

This is my first personal portfolio where I showcase my front-end development skills.

## 🚀 Features

- Fully responsive design (desktop, tablet, mobile)
- Adaptive navigation bar with a hamburger menu on smaller screens
- Language switcher (English / French)
- Smooth scrolling experience
- Scroll-based animations implemented with JavaScript
- Contact form powered by EmailJS (messages are sent directly to my email)

## 🛠️ Technologies Used

- HTML5
- CSS3 / SCSS
- JavaScript
- EmailJS
